# AI工作流：学习与调研指南

## 一、AI模型选择指南

### 1.1 选择标准
- 联网搜索能力
- 知识库更新及时性
- 知识整理能力
- 响应准确度

### 1.2 推荐模型
- **Deepseek-R1**（开源基座模型）
- **秘塔AI**（优秀的知识整理能力）
- 其他可选项：ChatGPT、Claude等
